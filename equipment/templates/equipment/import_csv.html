CSV Format Requirements

Your CSV file should have the following columns:

  • code (required) - Unique equipment code
  • name - Equipment name
  • description - Description
  • status - One of: active, inactive, maintenance, decommissioned
  • location - Physical location
  • parent_code - Code of parent equipment (optional)
  • category_name - Name of the category (optional)

The first row of the CSV should contain the column headers.
